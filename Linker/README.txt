In order to compile, please cd to the directory containing the Linker and module file and enter: 
	javac *.java

To run the program, please enter: 
	java Linker input output

where Linker is the name of the main method of the program, input is the directory address of the .txt file which is taken as the first argument, and output will serve as the file name and directory address for the outputted .txt file. (i.e. C:\Users\rc3009\Documents\input.txt)

	WARNING: If the name of the output .txt file corresponds to the name of another 
	file in the same directory, the previous file will be irrevocably overwritten.
