Programming Language: Java

Please place all inputs and the random files into the src folder.

In order to compile, please cd to the directory containing the Linker and module file and enter: 
	javac *.java

To run the program, please enter: 
	java Scheduler input
or for the verbose output enter:
	java Scheduler --verbose input

where Scheduler is the name of the main method of the program, input is the directory address of the .txt file which is taken as the first argument, or in the case that "--verbose" is the first argument, the second argument in the form: src\inputx.txt
